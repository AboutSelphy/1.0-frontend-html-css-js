# 1.0 – Frontend: HTML, CSS, and JavaScript

Welcome to **Part 1.0** of my structured web development relearning journey. This repository focuses on building web interfaces using **vanilla HTML, CSS, and JavaScript**—with no frameworks or libraries. The goal is to master the core web technologies that form the foundation of modern frontend development.

---

## 🧠 Purpose

- Relearn web development fundamentals from the ground up
- Write clean, semantic, and accessible HTML
- Style layouts using pure CSS (no frameworks or preprocessors)
- Add interactivity and DOM manipulation with plain JavaScript
- Understand the building blocks behind modern tools

---
